SQL 報表作品：採購/庫存

Overview

這個作品是什麼

我用 SQLite 建立小型採購/庫存資料模型與範例資料，並撰寫常見管理報表查詢（可直接執行）。

我想展示的能力

  • 把流程概念轉成可查詢的資料結構
  • 用 JOIN/彙總/Top-N 做管理報表
  • 查詢命名、拆解與註解保持可讀性

作品檔案

  • README
  • schema.sql
  • queries.sql
看查詢 看前端儀表板

如何執行

sqlite3 demo.db < schema.sql
sqlite3 demo.db < queries.sql

我已測試可正常輸出：採購總覽、月彙總、Top-N、庫存、低庫存、供應商統計。
